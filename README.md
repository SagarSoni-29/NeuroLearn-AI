# 🧠 NeuroLearn AI – Intelligent Personalized Study Companion

NeuroLearn AI is an AI-powered personalized study assistant designed to help students learn smarter using AI.

The platform combines document intelligence, AI chat, quiz generation, and notes summarization into one interactive learning environment.

---

## 🚀 Features

### 💬 AI Study Chatbot
- Ask study-related questions
- AI-generated structured responses
- Professional formatting

### ❓ Quiz Generator
- Generate quizzes (up to 5 questions)
- Topic-based learning
- Answer highlighting

### 📄 PDF / Document Chat
- Upload PDF and DOCX files
- Ask questions from uploaded documents
- Context-aware responses

### 📝 Notes Summarizer
- Convert long notes into concise summaries
- Important point extraction
- Student-friendly formatting

### 🔐 Authentication System
- User Login / Signup
- Password hashing using bcrypt
- Session management

### 🎨 Interactive Interface
- Animated login page
- Futuristic landing page
- Curtain entry effect
- Dynamic tab workspace
- Glassmorphism dashboard
